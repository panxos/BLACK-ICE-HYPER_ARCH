## üîÑ Personalizar Atajos

### Archivo de Configuraci√≥n

Los atajos se definen en:

```
~/.config/hypr/hyprland.conf
```

### Sintaxis

```bash
# Formato general
bind = MODIFIERS, KEY, ACTION, PARAMS

# Ejemplos
bind = SUPER, Return, exec, kitty
bind = SUPER SHIFT, Q, killactive
bind = SUPER, 1, workspace, 1
```

### Modificadores Disponibles

- `SUPER` - Tecla Windows/Super
- `SHIFT` - Shift
- `CTRL` - Control
- `ALT` - Alt
- `MOD4` - Super (alternativo)

### Acciones Comunes

- `exec` - Ejecutar comando
- `killactive` - Cerrar ventana activa
- `workspace` - Cambiar workspace
- `movetoworkspace` - Mover a workspace
- `togglefloating` - Alternar flotante
- `fullscreen` - Pantalla completa

### Ejemplo de Personalizaci√≥n

```bash
# Agregar atajo personalizado
bind = SUPER CTRL, T, exec, thunar

# Cambiar atajo existente
bind = SUPER, Q, exec, wlogout

# Atajo para script personalizado
bind = SUPER ALT, S, exec, ~/.config/bin/mi_script.sh
```
